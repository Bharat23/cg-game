# Berzerk - The winterfell edition
This game started off as a final project for Computer Graphics class and ended up here.


## To Run locally
- Clone the repo
```
    git clone <repo-url> && cd <repo-name>
```

- Install the dependencies
```
npm install
```

- Run the game
```
npm start
```
